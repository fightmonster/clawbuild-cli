# ClawBuild CLI

Release-only distribution for ClawBuild CLI.

Install:

```bash
npm install -g https://github.com/fightmonster/clawbuild-cli/releases/download/v0.2.1/clawbuild-cli.tgz
```

Verify:

```bash
clawbuild --version
clawbuild --help
```

Source repository:

```text
https://github.com/fightmonster/clawbuild-cli-src
```
